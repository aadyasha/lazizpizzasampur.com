# 🍕 lazizpizzasampur.com

## Overview

**lazizpizzasampur.com** is a restaurant website project designed for *Laziz Pizza*. The project focuses on building a clean, structured, and visually appealing web interface using modern web development practices. It emphasizes semantic HTML, responsive design, and user-friendly layout for an enhanced browsing experience.

## Features

* Structured and semantic HTML for better accessibility and SEO
* Responsive design optimized for desktop and adaptable to different screen sizes
* Clean and modern UI layout
* Well-organized CSS styling
* Easy navigation for users

## Tech Stack

* **HTML5** – for semantic and structured content
* **CSS3** – for styling and layout design

## Project Structure

```
lazizpizzasampur.com/
│── index.html
│── style.css
│── images/
│── assets/
```

## Getting Started

To run this project locally:

1. Clone the repository

   ```
   git clone https://github.com/your-username/lazizpizzasampur.com.git
   ```
2. Navigate to the project folder

   ```
   cd lazizpizzasampur.com
   ```
3. Open `index.html` in your browser
